## Sobre o Projeto

Este repositório organiza um conjunto de estudos e exercícios de JavaScript para quem está iniciando ou revisando os fundamentos da linguagem.

A estrutura do projeto acompanha um fluxo didático:

- Conceitos básicos de variáveis, tipos e operadores.
- Manipulação de arrays e compreensão de estruturas de repetição.
- Importação e exportação de módulos em JavaScript.
- Desafios práticos para consolidar lógica e raciocínio computacional.

O conteúdo é voltado para estudantes, desenvolvedores iniciantes e qualquer pessoa interessada em reforçar conhecimentos em JavaScript.

## Como Executar

### Abrir os arquivos

1. Abra o projeto no seu editor de código preferido (recomendado: Visual Studio Code).
2. Navegue até o diretório desejado.
3. Abra os arquivos `.js` para estudar o código e fazer alterações.

### Executar com Node.js

```bash
cd "d:\Dio\Formação JavaScript Dev\Aprenda a Programar do Zero com JavaScript\Arrays e Estruturas de Repetição em JavaScript\entendendo arrays"
node O_Que_e_Um_Array.js
```

> Os exemplos podem ser executados em qualquer arquivo `.js` do repositório usando `node <nome-do-arquivo>.js`.
